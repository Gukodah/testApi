## Getting Started

Tho run this project follow the instructions below

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gukodah/testApi.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Rename env.example inside the src folder to `.env`

5. Run server
   ```sh
   npm start
  ```
